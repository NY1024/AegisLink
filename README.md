# AegisLink — 灵盾

> Forging the Chain of Trust for AI Agents — 为每一个 AI Agent 铸造可信身份，让多智能体协作安全互联。

## 项目简介

AegisLink 是一套面向多 Agent 协作场景的身份认证与权限管控基础设施。

在 AI Agent 时代，工作流从传统的 `User → App` 演变为 `User → Agent A → Agent B → Service C` 的多级调用链，传统 IAM 体系的安全假设被彻底打破。AegisLink 为这条信任链中的每一个 Agent 铸造可验证的数字身份，以能力声明定义权限边界，以委托协议传递信任上下文，以实时策略引擎拦截越权行为，以全链路审计日志实现行为可追溯,为 AI Agent 的企业级落地筑牢安全底座。

### 核心能力

- 🔐 **身份签发与验证** — 为每个 Agent 签发唯一 Access Token，调用链中逐级验证请求来源
- 🎯 **能力声明与细粒度管控** — 基于 Capability Statement 精确定义每个 Agent 的操作边界
- 🤝 **委托授权与信任传递** — 支持 Agent 代表用户执行操作，自动计算 `用户权限  Agent 能力` 的有效权限交集
- 🛡️ **实时越权拦截** — 策略引擎在 Agent 间调用时强制校验权限，非法请求即时阻断
- 📋 **全链路审计追溯** — 记录每一次授权决策的完整上下文，支持调用链可视化追踪

### 演示场景

以飞书企业协作为背景，接入三个职责不同的 Agent：

| Agent | 职责 | 权限边界 |
|-------|------|----------|
| 飞书文档助手 Agent | 理解用户需求，编排任务，生成报告写入飞书文档 | 可委托其他 Agent，可写入文档 |
| 企业数据 Agent | 访问飞书通讯录、多维表格等企业内部数据 | 唯一拥有企业数据读取权限 |
| 外部检索 Agent | 抓取公开网页信息 | 仅限外部公开数据，无企业数据访问权 |
